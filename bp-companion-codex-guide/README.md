# BP Companion：Codex CLI 实施包

版本日期：2026-06-22

本实施包用于开发一个微信小程序，帮助高血压患者通过手动、语音或拍照快速记录血压，并完成运动与饮食任务，并允许患者授权家属查看完成情况。

本包包含：

- `QUICK_START.md`：从零到微信小程序上线的最短路径。
- `IMPLEMENTATION_GUIDE.md`：完整的产品、工程、测试、评审与发布方案。
- `AGENTS.md`：供 Codex CLI 自动读取的仓库级规则。
- `CODEX_PROMPTS.md`：分阶段复制给 Codex CLI 的任务指令。
- `REVIEW_CHECKLIST.md`：独立代码评审与发布检查清单。
- `.github/PULL_REQUEST_TEMPLATE.md`：Pull Request 模板。

建议先读 `QUICK_START.md`，随后让 Codex 按 `CODEX_PROMPTS.md` 的 Phase 0 开始。
